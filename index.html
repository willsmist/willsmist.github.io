上面的命令先进行 SSH 登录，使用密码或公钥验证，然后再使用 git 用户默认 shell 执行克隆操作。
所以用户 git 的默认 shell 必须能够执行命令，若设置为 /bin/false 则因为无法执行 git clone 操作而失败
